Notes

  • .table-responsive makes the table horizontally scroll when it overflows the container (Bootstrap only).
  • You can use breakpointed variants like .table-responsive-sm, -md, -lg, etc., to enable scrolling only below a specific breakpoint.
  • If you need vertical scrolling with a fixed height, wrap the table in a div with a fixed height and overflow-auto.
